----------Бэкапы----------
Бэкап - это резервная копия данных. Есть 2 команды: экспорт и импорт данных
Команда экспорта выполняет создание бэкапа, а команда импорта восстанавливает данные
на основе бэкапа.

Для создания бэкапа используйте:
pg_dump -U postgres -h localhost flask_shop >> flask_shop.sql

Для восстановления из бэкапа используйте:
psql -f demo_small.sql -U postgres
