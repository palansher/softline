# между сервером и клиентом происходит полноценный диалог

1. Сервер засыпает на строке server.accept().

2. Клиент подключается через client.connect().

3. Сервер просыпается, создает сокет sock, получает адрес клиента и сразу же засыпает на новой строке sock.recv(1024), ожидая текст.
4. Клиент отправляет строку "Привет, сервер! Как дела?" с помощью client.send() и сам засыпает на строке client.recv(), ожидая ответ.
5. Сервер принимает байты, декодирует их, выводит в консоль, отправляет ответ обратно и закрывает сокет.
6. Клиент просыпается, получает ответ от сервера, выводит его на экран и завершает работу.
